Currently on the network

Diesel Multiple Units

Irish Rail

22000 Class (ICR)

Built by Hyundai Rotem and Tokyu between 2007 and 2012 to replace the decaying British Mk2 and Cravens coaches, they entered service that year and were deployed on intercity and some commuter services, and operate some Enterprise services to Belfast making them the backbone of all Irish rail services.

29000 Class

Built by CAF between 2002 and 2005, you can mostly find these trains on Dublin area commuter services and sometimes the Enterprise if an ICR set is unavailable. They are due to be replaced by the 90000 class Dart+ units soon and will probably be cascaded to replace the aging 2600 DMUs in Cork.

2800 Class

Built by Tokyu in 2000, these are primarily used on western railway corridor services out of Limerick, and occasionaly sub in for a 2600 for Cork commuter services if one is unavailable. The cab end gangways were recently removed as they were unused and the CCTV installed would stop it from being used anyway

2600 Class

Built by Tokyu between 1993-1994, the 2600s were the first modern Irish DMU and originally served Dublin "Arrow" commuter services before later moving to Cork.

Translink NI Railways

3000 Class

Built by CAF between 2004 and 2005 to replace the aging 80 class, they operate on all NIR services and some units also operate on the Enterprise to help with the hourly timetable

4000 Class

Built by CAF between 2011 and 2012, these replaced the final 450 class trains. These trains were not cleared for working past the border unlike the 3000s and operate only in Northern Ireland.
